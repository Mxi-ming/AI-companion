### 🤖 AI智能伴侣

这是一个基于 Python 和 Streamlit 开发的 AI 聊天伴侣应用。它利用大语言模型（当前配置为 DeepSeek）模拟真实的伴侣角色，提供沉浸式的聊天体验，并支持多会话管理和历史记录保存。

### 🚀 快速开始

**前提条件**
- Python 3.8 或更高版本
- 已注册 DeepSeek API 账号并获取 API Key

**安装与运行**
1.  克隆项目：
        git clone [你的仓库地址]
    cd AI-companion
    

2.  安装依赖：
    确保你已安装 Streamlit 和 OpenAI 库：
        pip install streamlit
        pip install openai
    

4.  配置环境变量：
    右键此电脑，点击属性，再点击高级系统设置，接着点击环境变量，新建变量名为： DEEPSEEK_API_KEY，变量值则填入你的 API 密钥
    

6.  启动应用：
        streamlit run AICompanion.py

### ⚙️ 功能特性

- 沉浸式角色扮演：你可以自定义伴侣的名称和人设（如“活泼开朗”、“高冷忧郁”等），AI 会严格遵循你设定的人设进行对话。
- 多会话管理：
    - 新建会话：点击侧边栏按钮即可开启全新的聊天记录。
    - 历史加载：侧边栏自动列出所有历史会话，支持一键加载。
    - 会话删除：支持删除不再需要的历史记录。
- 本地持久化存储：所有的聊天记录和会话状态都会自动保存到本地的 sessions 文件夹中（JSON 格式），关闭浏览器后数据不会丢失。
- 流式输出：消息逐字显示，模拟真实打字效果，提升交互体验。

### 🧠 核心逻辑

该应用的核心在于 system_prompt 的设计。系统会将用户设定的角色信息（名称、性格）注入到系统指令中，并强制要求 AI 遵循以下规则：

1.  完全代入角色：禁止以 AI 身份回复。
2.  回复简短：像微信聊天一样，每次只回一条消息，避免长篇大论。
3.  匹配语言：自动适配用户的语言风格。
4.  情感表达：鼓励使用 emoji (如 ❤️🌸) 增加亲密度。
5.  禁止描述性文字：不输出动作或场景描写，只输出对话内容。

### 📂 文件结构

项目采用清晰的模块化结构，便于维护和扩展：

- `AICompanion.py`：主程序入口，负责启动应用和核心逻辑调度。
- `sessions/`：存放历史会话记录的文件夹，每个会话以 JSON 文件形式保存。
  - `*.json`：单个会话的 JSON 数据文件。
- `resources/`：资源文件夹，存放应用所需的静态资源。
  - `logo.png`：应用 Logo 图片。

### 🛠️ 技术栈

- Streamlit：用于构建快速、美观的 Web 界面。
- OpenAI Python SDK：用于调用大语言模型 API（此处兼容 DeepSeek API）。
- Python JSON：用于本地数据的序列化和持久化存储。
- DeepSeek API：作为后端的大语言模型推理服务。
